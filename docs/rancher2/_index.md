## 概述

Rancher 2.x 目前有三个大版本：2.3.x、2.4.x 和 2.5.x，每个大版本中有多个小版本，如 2.3.0、2.4.0 和 2.5.0。由于 2.5.x 系列变化较大，我们会将其单独列出。详情请参考[Rancher 2.5 版本说明](/docs/rancher2.5/_index)。

本文按照版本的发布时间由新到旧罗列了 Rancher 2.3.x 和 2.4.x 中所包含的所有小版本及对应的版本说明。
