Rancher 支持具有各种卷插件的持久存储。但是，在使用这些插件将持久性存储绑定到您的工作负载之前，您必须先配置存储本身，无论它是来自云提供商还是由您自行管理。

Rancher 提供了一些关于如何配置常用的存储服务的文档，供您参考：

- [NFS](/docs/rancher2/cluster-admin/volumes-and-storage/examples/nfs/)
- [vSphere](/docs/rancher2/cluster-admin/volumes-and-storage/examples/vsphere/)
- [亚马逊 EBS](/docs/rancher2/cluster-admin/volumes-and-storage/examples/ebs/)
