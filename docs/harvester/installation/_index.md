目前有两种方式可以安装 Harvester：您可以通过 ISO mode 或 App mode 安装 Harvester，其中 App mode 适用于测试和开发环境，不建议在生产环境中使用。

请参考以下链接，获取对应的安装说明：

- [ISO 模式](/docs/harvester/installation/iso-mode/_index)
- [PXE 模式](/docs/harvester/installation/pxe-mode/_index)
