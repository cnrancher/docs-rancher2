## 上传镜像

要在 **Images** 页面中上传虚拟机镜像，请输入可从群集访问的 URL。图像名称将使用 URL 地址的文件名自动填充。您可以根据需要随时修改。

目前，我们支持 qcow2、raw 和 ISO 镜像。

描述和标签不是必填项。
